mode

Extract

Print the extracted content and stop. No summary call.

Usage

summarize --extract "https://example.com/article"
Good for piping into your own tooling.

Helpful flags

  • --json — emit a single JSON object.
  • --verbose — show which extractor ran and why.
  • --timeout — tune crawling budget (2m default).
  • --firecrawl off|auto|always — choose the fallback strategy.
  • --format md|text — choose extracted output format.
  • --markdown-mode off|auto|llm|readability — for non-YouTube URLs with --format md, control HTML→Markdown conversion (LLM) + fallbacks.
  • --preprocess off|auto|always — controls markitdown usage (default auto).
