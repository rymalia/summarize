OpenAI

Use OpenAI by setting OPENAI_API_KEY and picking a model via --model.

Notes

  • Some modes (like --extract) don’t need an LLM at all.
  • When output is used downstream, prefer --json and pin --model.
If your extraction is flaky, flip on --verbose and inspect which path was taken (HTML vs Firecrawl vs LLM Markdown).
