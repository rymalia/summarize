Chrome extension

Summarize ships a Chrome Side Panel extension backed by a tiny local daemon. It streams Markdown summaries for the active tab and supports auto-summarize on navigation.

Setup

  • npm i -g @steipete/summarize or brew install steipete/tap/summarize
  • Build extension: pnpm -C apps/chrome-extension build
  • Load unpacked: apps/chrome-extension/.output/chrome-mv3
  • Open side panel → copy token command → run summarize daemon install --token <TOKEN>
  • Verify: summarize daemon status

Dev

  • Install daemon for dev: pnpm summarize daemon install --token <TOKEN> --dev
  • Dev server: pnpm -C apps/chrome-extension dev

Troubleshooting

  • Daemon unreachable: summarize daemon status
  • Logs: ~/.summarize/logs/daemon.err.log
  • Token mismatch: update token in side panel and Save
The daemon listens on 127.0.0.1:8787 only and is token-protected.
