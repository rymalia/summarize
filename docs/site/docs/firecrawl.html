extractor

Firecrawl

Used as a fallback when HTML extraction looks blocked or too thin — and as a preferred Markdown source in extract mode (when configured).

Key

  • Set FIRECRAWL_API_KEY.
  • Control behavior with --firecrawl off|auto|always.

Extract + Markdown

  • --extract prints the extracted content.
  • --extract --format md outputs Markdown for non-YouTube URLs.
If you only want plain text: use --extract --format text.
